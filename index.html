Introduction

HTML (Hypertext Markup Language) is the standard language for creating web pages. It allows developers to structure and display content through elements, tags, and attributes that describe different parts of a web page, like headings, paragraphs, images, and links.

HTML is used in conjunction with CSS (for styling) and JavaScript (for functionality) to create fully interactive and responsive websites. Each HTML document is essentially a set of nested elements, creating a tree structure known as the Document Object Model (DOM).

  • The (!DOCTYPE html) declaration defines that this document is an HTML5 document.
  • The (html) element is the root element of an HTML page.
  • The (head) element contains meta information about the HTML page.
  • The (title) element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).
  • The (body) element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The (h1) element defines a large heading.

Example Code:


            <!DOCTYPE html>
            <html>
            <head><title>Hello World</title></head>
            <body><h1>Hello, World!</h1></body>
            </html>
Basic Structure of an HTML Document

A standard HTML document includes a DOCTYPE declaration, followed by html, head, and body sections.The head section contains metadata, links to stylesheets, and scripts, while the body section contains the content visible to the user.

The basic structure of an HTML document includes several key elements that help define its content and layout. At the top, the <!DOCTYPE html> declaration is used to specify that the document is an HTML5 document, ensuring modern browser compatibility. The entire content is wrapped in the <html> tag, which also contains a lang attribute to define the language of the document. Inside the <html> tag, the <head> section contains meta-information like character encoding, the document's title, and links to external resources such as stylesheets and scripts. The actual content that users see is placed inside the <body> section, which can include headings, paragraphs, images, links, and other elements to display the page’s information. Additionally, the use of tags like <header>, <main>, and <footer> helps organize content into sections, providing a structured layout for better accessibility and readability.

Components:

  • DOCTYPE: Specifies the HTML version (typically HTML5).
  • Head: Contains metadata and links to external resources.
  • Body: Contains the main content displayed to users.

Example Code:


          <!DOCTYPE html>
          <html>
          <head>
            <title>Page Title</title>
          </head>
          <body>
            <h1>My First Heading</h1>
            <p>My first paragraph.</p>
          </body>  
          </html>
Text Formatting

HTML includes a variety of tags to format text. These tags are used to define elements like headings, paragraphs, bold, italics, and lists. This helps structure text content, making it easy to read and visually appealing.

In HTML, text formatting allows you to modify the appearance and style of text to improve readability and presentation. The <b> tag is used to bold text, while the <i> tag italicizes text. If you want to underline text, you can use the <u> tag. For making text stand out more semantically, the <strong> tag is used for important text, and the <em> tag is used for emphasized text. Additionally, the <mark> tag highlights text, which is useful for drawing attention to specific words or phrases. Combining these tags with CSS properties can further enhance text styling by adjusting font size, color, and alignment, providing a more visually appealing user experience.

Examples of text tags:

  • Headings: <h1>, <h2>, etc.
  • Paragraph: <p>
  • Bold: <b> or <strong>
  • Italic: <i> or <em>

Example Code:


            <p><strong>Bold text</strong> and <em>italic text</em>.</p>
Links and Images

Links in HTML: In HTML, links are created using the <a> (anchor) tag, which allows users to navigate from one webpage to another. The href attribute is used to specify the destination of the link, which could be another webpage, a section of the same page, or even an external resource. Links can be styled using CSS to enhance their appearance, such as changing color or adding hover effects. For example, <a href="https://example.com">Click here</a> creates a link to an external website. Links are essential for connecting different pages and resources on the web.

Images in HTML: Images are included in HTML documents using the <img> tag. The src (source) attribute is used to specify the location of the image file, and the alt attribute provides alternative text in case the image cannot be displayed. Images can be resized using CSS properties like width, height, or by setting the max-width to 100% for responsive layouts. For example, <img src="image.jpg" alt="Description of image"> embeds an image into the page. Proper use of images and their descriptions enhances accessibility and user experience.

Attributes:

  • href (for <a>): The URL to which the link points.
  • src (for <img>): The path to the image file.

Example code:


            <a href="https://example.com">Visit Example</a>
            <img src="image.jpg" alt="Example Image">
Lists

HTML provides support for both ordered and unordered lists, enabling easy structuring of items like bullet points or numbered lists.

In HTML, lists are used to group a set of related items. There are three types of lists: unordered lists, ordered lists, and description lists. Each list type serves a specific purpose, and they are represented using different HTML tags.

  • Unordered List: An unordered list is used when the order of the list items doesn’t matter. It is created using the <ul> (unordered list) tag, and each item inside the list is wrapped with the <li> (list item) tag.

    • Example code:

    
              <ul>
                <li>Apples</li>
                <li>Oranges</li>
                <li>Bananas</li>
              </ul>
  • Ordered List: An ordered list is used when the order of the items is important. It is created using the <ol> (ordered list) tag. The items inside the list are also wrapped with the <li> tag. The browser will automatically number the items in an ordered list:

    • Example code:

    
              <ol>
                <li>Step 1</li>
                <li>Step 2</li>
                <li>Step 3</li>
              </ol>
  • Description List: A description list is used to define a list of terms and their descriptions. It is created using the <dl> (description list) tag, with each term wrapped in the <dt> (description term) tag and its description in the <dd> (description definition) tag. This is useful for glossaries or definitions:

    • Example code:


            <dl>
              <dt>HTML</dt>
              <dd>Hypertext Markup Language</dd>
              <dt>CSS</dt>
              <dd>Cascading Style Sheets</dd>
            </dl>
